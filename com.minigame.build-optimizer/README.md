# MiniGame Build Optimizer

WeChat MiniGame build optimizer: diagnostics, one-click optimization, asset bundling, font subsetting, and shader variant analysis.

## 适用范围

本包为 **MiniGame Toolchain 免费版** 的一部分，包含 编辑器工具与运行时组件。

## 安装

1. 解压分发包。
2. 在 Unity Package Manager 中选择 **Add package from disk...**。
3. 选择本文件夹（包含 `package.json` 的目录）。

## 依赖

- `com.minigame.core`: `0.1.2`
