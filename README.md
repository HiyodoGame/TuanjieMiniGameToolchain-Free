# MiniGame Toolchain 免费版 UPM 包合集

- **版本**: 0.1.2
- **生成日期**: 2026-07-13

## 包含的包

- `com.minigame.build-optimizer`
- `com.minigame.core`
- `com.minigame.performance-suite`
- `com.minigame.starter-kit`

## 安装方式

1. 解压本 bundle。
2. 在 Unity Package Manager 中依次选择 **Add package from disk...**，然后选择每个包的目录。
3. 或直接将各包目录复制到项目的 `Packages/` 文件夹。
