# MiniGame Toolchain Core

Shared core library for the MiniGame Toolchain product matrix (Build Optimizer, Performance Suite, Starter Kit, Builder Pro).

## 适用范围

本包为 **MiniGame Toolchain 免费版** 的一部分，包含 核心共享库。

## 安装

1. 解压分发包。
2. 在 Unity Package Manager 中选择 **Add package from disk...**。
3. 选择本文件夹（包含 `package.json` 的目录）。
